  regex; `src/shared/redact.ts:8` now allows `-`/`_` and explicitly covers
  `sk-ant-api03-…`. Fixed as alignment item A2 (commit `7f97e6b`).
- **Skill-injection sanitizer is bypassable** — real, but already tracked as

  once per finish** — real patterns, but micro-optimizations in a network-bound
  agent loop dominated by multi-second LLM/browser round-trips. Poor leverage;
  the readability/dedup angle is already inside alignment item **C3**.
- **`displaySafeUrl` unwired, `cloneJson` / `HELPER_PREAMBLE` dead code, verdict
  module extraction, three identical signal interfaces, agent↔eval import cycle,
  experiments module untested** — all already in the alignment plan (**C4, C3,
  C2, A10/A13**). The agent↔eval allowlist in `scripts/architecture-check.mjs`
  is intentional and slated to be burned to zero (C2).
- **Contract-reprompt does not push a `thought-summary` guidance event**
  (`finish-flow.ts:173`) — the `contract-check` event *is* pushed to the trace
  (lines 166-172) and the step-count increment mirrors the extraction-reprompt
